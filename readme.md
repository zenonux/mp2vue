## 安装
```npm i -g mp2vue```

## 使用
```
cd mp-dir               #进入小程序项目目录
mp2vue                  #默认参数'./pages'
mp2vue -p ./pages       #自定义解析目录
```

## wxml转换
    基本的标签类view,block,text,scroll-view,image,navigator
    基本的属性类wx:if,wx:elif,wx:else,wx:for,wx:key,src,bindtap,catchtap,bindinput,hover-class,class,url
    单位转换 2rpx = 1px,1rpx=1px
## wxss转换
    单位转换 2rpx = 1px,1rpx=1px
## less转换
    单位转换 2rpx = 1px,1rpx=1px
